Description project_c_samples.parquet

The file contains pre-labeled place records. The open and closed status is defined via the column "open", where 1 := "place is open", 0 := "place is closed". The following columns are 
present: ['id', 'geometry', 'bbox', 'type', 'version', 'sources', 'names',
       'categories', 'confidence', 'websites', 'socials', 'emails', 'phones',
       'brand', 'addresses', 'open'].

Further information on the schema can be found here: https://docs.overturemaps.org/schema/reference/places/place/
